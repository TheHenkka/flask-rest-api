Welcome to webstore API

Usage:

GET 'http://127.0.0.1:5000/cart/[cart_id]'

-API fetches cart data from database using cart ID.

PUT 'http://127.0.0.1:5000/cart, {'items': '[1,2,3,4]', 'country': 'country_name'}'

-API saves new cart to database and sends ID back as response.

DELETE 'http://127.0.0.1:5000/cart/[cart_id]'

-API deletes data from database and returns HTTP code

PATCH 'http://127.0.0.1:5000/cart/[cart_id], {'items': '[1,2,3,4]', 'country': 'country_name'}'

-API updates and saves data, and returns HTTP code
